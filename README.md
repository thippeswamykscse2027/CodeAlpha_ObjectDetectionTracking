# Object Detection and Tracking using YOLOv8

## Overview

This project performs real-time object detection and tracking using the YOLOv8 deep learning model and OpenCV. The system can detect multiple objects from a webcam feed and assign unique tracking IDs to each detected object.

## Features

* Real-time webcam video processing
* Object detection using YOLOv8
* Object tracking with unique IDs
* Bounding boxes and labels
* Live visualization

## Technologies Used

* Python
* OpenCV
* YOLOv8 (Ultralytics)
* NumPy

## Installation

Clone the repository:

git clone https://github.com/thippeswamykscse2027/CodeAlpha_ObjectDetectionTracking.git

Move into the project folder:

cd CodeAlpha_ObjectDetectionTracking

Install dependencies:

pip install -r requirements.txt

## Run the Project

python object_tracking.py

## Project Workflow

1. Capture frames from webcam.
2. Detect objects using YOLOv8.
3. Track detected objects.
4. Assign tracking IDs.
5. Display results in real time.

## Output

The application displays:

* Bounding boxes around detected objects
* Object labels
* Tracking IDs
* Live webcam feed
